# SLC-Tech Page

This project is a fork of [hugoplate theme](https://github.com/zeon-studio/hugoplate). 

## How to Run in Dev

Use the `run.sh` script to run the webiste. Head over to `http://localhost:1313` to use the website.

## How to Deploy 

- Run `deploy.sh` a and stop it using `stop.sh` (when required)
- The final website will be hosted in port 81. Please configure your nginx to use it.
